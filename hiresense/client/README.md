# HireSense – AI-Based Recruitment Platform 🚀

HireSense is a full-stack web application designed to help job seekers manage their profiles and connect with opportunities through a modern recruitment platform.

---

## ✨ Features

- User Registration & Login
- Secure Authentication
- Candidate Profile Management
- Profile Update Functionality
- MongoDB Database Integration
- Responsive UI
- RESTful API Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- JavaScript
- CSS

### Backend
- Flask (Python)
- MongoDB
- PyMongo
- bcrypt
